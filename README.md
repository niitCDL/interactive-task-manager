## 🚀 快速开始

### 在线体验
直接打开 `index.html` 文件即可开始使用！

### 本地部署
```bash
# 克隆仓库
git clone https://github.com/niitCDL/interactive-task-manager.git

# 进入项目目录
cd interactive-task-manager

# 使用任意HTTP服务器运行
# 方法1：使用Python
python -m http.server 8000

# 方法2：使用Node.js
npx serve .

# 方法3：使用Live Server (VS Code扩展)
# 右键点击index.html -> "Open with Live Server"
```

然后在浏览器中访问 `http://localhost:8000`

## 🔧 自定义配置

### 主题颜色
在 `styles/main.css` 中修改 CSS 变量：

```css
:root {
    --accent-primary: #667eea;    /* 主色调 */
    --accent-secondary: #764ba2;  /* 辅助色 */
    --success: #48bb78;           /* 成功色 */
    --danger: #e53e3e;            /* 危险色 */
    --warning: #ed8936;           /* 警告色 */
}
```

### 粒子效果
在 `scripts/app.js` 中调整粒子参数：

```javascript
// 粒子数量
for (let i = 0; i < 30; i++) {
    this.particles.push(this.createParticle(x, y));
}

// 粒子颜色
const colors = ['#ff6b6b', '#4ecdc4', '#45b7d1', '#96ceb4'];
```
